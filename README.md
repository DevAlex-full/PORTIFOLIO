# Portfólio - Alexander Bueno Santiago

Um portfólio web moderno e responsivo desenvolvido com tecnologias front-end, apresentando projetos e habilidades de desenvolvimento web.

## 🌟 Visão Geral

Este portfólio foi criado para apresentar meus projetos, habilidades e experiência como desenvolvedor Fron-End. O site conta com design moderno, animações fluidas e uma experiência de usuário otimizada.

### 🔗 Links Importantes

- **Demo ao vivo**: [https://portifoliodevalex.netlify.app/]
- **Repositório**: [https://github.com/DevAlex-full/PORTIFOLIO)

## 🚀 Funcionalidades

- Design responsivo e mobile-first
- Navegação suave entre seções
- Animações e efeitos visuais modernos
- Sistema de fallback inteligente para imagens
- Formulário de contato funcional
- Otimização de performance com lazy loading
- Tema escuro com gradientes roxos
- Acessibilidade aprimorada

## 🛠️ Tecnologias Utilizadas

- **JavaScript ES6+** - Interatividade e funcionalidades dinâmicas

- **Google Fonts** - Tipografia

## 🔧 Instalação e Uso

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/DevAlex-full/portfolio.git
   cd portfolio
   ```

2. **Abra o projeto**:
   - Abra o arquivo `index.html` em seu navegador
   - Ou use um servidor local como Live Server (VS Code)

3. **Para desenvolvimento**:
   ```bash
   # Com Python
   python -m http.server 8000
   
   # Com Node.js
   npx http-server
   ```

## 🎯 Recursos Técnicos

### Performance
- Lazy loading de imagens
- Throttling em eventos de scroll
- Otimização de animações com requestAnimationFrame
- Preload de recursos críticos

### Acessibilidade
- Navegação por teclado
- Skip links
- Alt texts descritivos
- Contraste adequado
- Semântica HTML apropriada

### Compatibilidade
- Suporte a navegadores modernos
- Fallbacks para funcionalidades não suportadas
- Polyfills quando necessário

## 🎨 Customização

### Cores
As cores principais podem ser alteradas no arquivo `main.css`:

```css
:root {
    --primary-color: #8b5cf6;    /* Roxo principal */
    --secondary-color: #a855f7;  /* Roxo secundário */
    --accent-color: #c084fc;     /* Roxo de destaque */
    /* ... outras variáveis */
}
```

### Conteúdo
- Edite o arquivo `index.html` para alterar textos
- Substitua as imagens na pasta `src/imagens/`
- Atualize os links de projetos e redes sociais
